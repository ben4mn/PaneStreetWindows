## [0.6.3]

### Added

- **Find in Terminal button** — A search button (⌕) is now visible in every pane header.
  Click it or press `Ctrl+Shift+F` to open an inline search bar within that terminal.
  Navigate matches with the ▲/▼ buttons or `Shift+Enter` / `Enter`. Press `Escape` to close.

- **Command Palette** — Press `Ctrl+P` to open a fuzzy-search overlay listing every
  available command and all open sessions. Type to filter, use `↑`/`↓` to navigate,
  `Enter` to execute, `Escape` to dismiss. The shortcut can be rebound in Settings → Keys.

- **Broadcast Input** — Each pane header now has a broadcast toggle (⊕). Enable it on
  two or more panes and keystrokes typed in any broadcast-enabled pane are mirrored to
  all other broadcast-enabled panes simultaneously. The button turns accent-colored when
  active. Useful for running the same command across multiple repos, environments, or
  shells at once.
